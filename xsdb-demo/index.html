Demo of XSDB

This page demonstrates how Cross-Site Document Blocking (XSDB) works. Please see one of the following resources for more information about XSDB:

  • Design document
  • Intent to ship

Press the button below to add an <img src="https://www.chromium.org/"> element that tries to sneak a cross-origin text/html document into memory of a renderer process hosting another origin.

The http request needed by the inserted img element should be blocked by XSDB and there should be a console message saying: Blocked current origin from receiving cross-site document at https://www.chromium.org/ with MIME type text/html.
